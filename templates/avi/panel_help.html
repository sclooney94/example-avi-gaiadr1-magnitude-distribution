About this AVI

This is a relatively Density Map AVI which demonstrates the following:

  • AVI service usage
  • Job queuing
  • Pipelines

Below is a list of some of the important files in this AVI

Density Map AVI contents:
Filename Description Related
urls.py Defines the URLs of the AVI views.py
views.py Defines functions which handle HTTP requests. The AVI job is created and started in 'run_query()'. tasks.py
tasks.py Defines the AVI pipeline models.py
models.py Defines the database models which store the job parameters. The parameters stored in these models are used in the pipeline. tasks.py
